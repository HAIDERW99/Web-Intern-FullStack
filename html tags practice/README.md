# DevAcademy – HTML5 Semantic Structure Demo

A clean, single-page website built with **pure HTML5 only** — no CSS frameworks, no JavaScript libraries. The goal is to demonstrate core HTML5 semantic and structural elements in a real-world layout.

---

## What's Inside

| Element | Usage |
|---|---|
| `<header>` | Site branding and primary navigation |
| `<nav>` | Anchor links — Home, Articles, Join Us, Contact |
| `<main>` | Wraps all primary page content |
| `<section>` | Two sections: Featured Articles and Student Registration |
| `<article>` | Self-contained blog post with heading, image, and lists |
| `<aside>` | Sidebar with mission statement and quick links |
| `<footer>` | Copyright notice and policy links |
| `<form>` | Registration form with multiple input types |
| `<div>` / `<span>` | Layout grouping and inline keyword highlight |

---

## File Structure

```
Html tags/
└── index.html    # The entire page — markup + embedded CSS
└── README.md
```

---

## Features

- **Semantic HTML5** — proper use of landmark elements for accessibility and SEO
- **Heading hierarchy** — `<h1>` → `<h2>` → `<h3>` used in correct order
- **Both list types** — `<ul>` for benefits, `<ol>` for a step-by-step learning path
- **Accessible form** — `<label>` tags tied to inputs via `for`/`id`; includes `text`, `email`, and `password` inputs plus a `<textarea>`
- **Inline styling only** — a `<style>` block in `<head>`, zero external dependencies

---

## How to Run

No build step needed. Just open the file in any browser:

```
double-click index.html
```

Or right-click → *Open with* → your preferred browser.
